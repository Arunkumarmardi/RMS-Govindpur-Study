üíæ Computer Memory

Memory stores data and instructions. Primary memory (RAM/ROM) is used during processing; secondary memory (HDD/SSD) stores long-term data.

  • RAM: Volatile, fast.
  • ROM: Non-volatile, stores firmware.
  • Cache: Very fast memory close to CPU.
