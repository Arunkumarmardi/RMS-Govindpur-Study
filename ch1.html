üíª What is a Computer?

Overview: A computer is an electronic device that accepts data (input), processes it according to instructions, stores results, and produces output. It performs tasks using programs and helps us solve problems quickly and accurately.

Components & Types

  • CPU: Processes instructions and performs calculations.
  • Memory: RAM and ROM store data temporarily and permanently.
  • Input Devices: Keyboard, Mouse, Scanner.
  • Output Devices: Monitor, Printer, Speaker.

Uses of Computers

  • Education, communication, entertainment, research, government, business.
