Chapter 7 — Input & Output Devices

⌨️ Input and Output Devices

Input devices send data to the computer; output devices display or produce results. Some devices are combined (touchscreen).

  • Input: Keyboard, Mouse, Microphone, Scanner, Webcam
  • Output: Monitor, Printer, Speaker, Projector

इनपुट डिवाइस कंप्यूटर को डेटा भेजते हैं; आउटपुट डिवाइस परिणाम दिखाते/प्रदान करते हैं। कुछ डिवाइस संयुक्त होते हैं (जैसे टचस्क्रीन)।
