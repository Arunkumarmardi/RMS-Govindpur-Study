ЁЯТ╗ Chapter 1: Introduction to Computers

Overview: A computer is an electronic device that accepts data (input), processes it as per instructions, stores information, and produces meaningful output. Modern computers can perform billions of operations per second and are used in nearly every field.

What is a Computer?

Computers take raw data and transform it into useful information through a sequence: Input тЖТ Process тЖТ Output тЖТ Storage. They follow programs (sets of instructions) to complete tasks.

Types of Computers

  • Personal Computers (PCs): Desktops and laptops used by individuals.
  • Tablets & Smartphones: Portable touch-based devices.
  • Servers: Powerful machines that provide services to other computers over a network.
  • Supercomputers: Extremely fast machines used for scientific calculations.

Components of a Computer

  • CPU (Central Processing Unit): The brain that performs calculations and executes instructions.
  • Memory: RAM and ROM for temporary and permanent storage of instructions/data.
  • Input Devices: Keyboard, mouse, scanner.
  • Output Devices: Monitor, printer, speakers.

Common Uses

  • Education: e-books, tutorials, assignments.
  • Communication: Email, messaging, video calls.
  • Entertainment: Games, movies, music.
  • Work: Documents, spreadsheets, presentations.

ЁЯФз Chapter 2: Hardware & Software

Overview: Hardware are the physical parts of a computer you can touch. Software is a set of instructions (programs) that tell the hardware what to do.

Hardware

Includes internal components like the motherboard, CPU, RAM, hard disk, and external devices like monitor, keyboard, mouse, printers. Hardware components have electrical and mechanical parts.

Software

Programs and applications that run on hardware. Software categories:

  • System Software: Operating System (OS) тАФ manages hardware and software resources (e.g., Windows, Linux).
  • Application Software: Programs to perform specific tasks (e.g., MS Word, Calculator, Games).
  • Utility Software: Tools to maintain the computer (e.g., antivirus, disk tools).

How Hardware and Software Work Together

Software provides instructions; CPU executes them using hardware. For example, a word processor (software) uses keyboard input (hardware) to enter text and displays it on the monitor (hardware).

ЁЯТ╛ Chapter 8: Computer Memory

Overview: Memory in computers stores instructions and data. It is divided mainly into primary (fast, temporary) and secondary (slower, permanent) memory.

Primary Memory

  • RAM (Random Access Memory): Volatile memory used by the CPU to store data currently in use. Data is lost when power goes off.
  • ROM (Read Only Memory): Non-volatile memory permanently storing important instructions like BIOS/firmware.
  • Cache Memory: Very small and fast memory located near the CPU to speed up operations.

Secondary Memory

Includes Hard Disk Drives (HDD), Solid State Drives (SSD), optical disks, and external drives. These store data permanently until deleted.

Why Memory Matters

More RAM helps run more applications simultaneously. Faster storage (SSD) improves boot times and application loading.

ЁЯТ╜ Chapter 9: Storage Devices

Overview: Storage devices keep data permanently. They vary by capacity, speed, portability, and durability.

Types of Storage

  • Hard Disk Drive (HDD): Magnetic disks, large capacity, cost-effective, mechanical parts.
  • Solid State Drive (SSD): No moving parts, faster read/write, more expensive per GB than HDD.
  • USB Flash Drives: Portable, used for quick file transfer.
  • Optical Disks (CD/DVD): Used for music, videos, and backups (less common now).
  • Memory Cards: Used in cameras and phones (SD cards, microSD).
  • Cloud Storage: Online services (Google Drive, OneDrive) accessible from any device with internet.

Choosing Storage

For speed and performance choose SSD. For large, inexpensive storage choose HDD. For portability choose USB drives or cloud storage.

ЁЯза Chapter 10: Computer Languages

Overview: Computer languages let developers write instructions that computers can execute. They range from low-level machine code to high-level languages.

Machine Language

The lowest level language consisting of binary digits (0 and 1). It is directly understood by the CPU but hard for humans to read/write.

Assembly Language

Uses symbolic instructions (mnemonics) instead of binary. An assembler converts assembly to machine code.

High-Level Languages

Languages like Python, Java, C++ are closer to human language. They allow building complex applications and are translated using compilers or interpreters.

Web Languages

HTML, CSS, and JavaScript are used to create webpages: HTML for structure, CSS for styling, and JavaScript for interactivity.
