Hardware - physical components you can touch (CPU, motherboard, monitor). Software - programs and instructions that run on hardware (operating systems, applications).

Software Types

  • System Software (Operating System)
  • Application Software (Word processors, browsers)
  • Utility Software (Antivirus, disk tools)
